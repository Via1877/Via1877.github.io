修改themes

Posted on 2018-03-09 | In technology

1,选择主题

我用的NexT

NexT使用文档http://theme-next.iissnan.com/

首先下载主题，定位在Hexo目录

git clone git@github.com:theme-next/hexo-theme-next.git

2,启用主题

在Hexo目录下，有一个配置文件_config.yml,被称为站点配置文件。
在next主题文件夹内，还有一个_config.yml配置文件，被称为主题配置文件。
启用主题时，打开站点配置文件，找到theme,将值修改为 next

theme:next

3,主题配置

a.外观选择
NexT有四种不同的外观，Muse、Mist、Pisces、Gemini

Muse - 默认 Scheme，这是 NexT 最初的版本，黑白主调，大量留白
Mist - Muse 的紧凑版本，整洁有序的单栏外观
Pisces - 双栏 Scheme，小家碧玉似的清新
Gemini - 双栏 Scheme，(在我看来和Pisces差不多)
更换Scheme，在主题配置文件中找到Scheme，将你需要的Scheme前面的#除去即可，如：我正在使用的Mist

# Schemes
#scheme: Muse
scheme: Mist
#scheme: Pisces
#scheme: Gemini

b.语言设置
编辑站点配置文件，将language设置成需要。例如我选择得简体中文

language: zh-Hans

c.设置菜单
主题配置文件中找到menu关键字，将#去掉即可，注意后面的路径。
图标使用的是Font Awesome提供的。

home: / || home
about: /about || user
#tags: /tags/ || tags
categories: /categories || th
archives: /archives || archive
#schedule: /schedule/ || calendar
#sitemap: /sitemap.xml || sitemap
#commonweal: /404 || heartbeat

d.设置头像
编辑主题配置文件，搜索avatar关键字，修改路径为头像链接地址，网络也可

avatar: /images/avatar.jpg
avatar: http://example.com/avatar.png

e.背景图片
next主题目录下/css/_custom/custom.styl,修改url。

body { 
    background-image: url(/images/bg.jpg);
    background-attachment: fixed;
    background-repeat: no-repeat;
}
